## Vektors

![](/blog/assets/top-design-resources-jan-2021/img-1.jpg)

[**https://www.vektors.pro**](https://www.vektors.pro/)

‍[Vektors](https://www.vektors.pro/) is the illustrations aggregator. All illustrations of this resource are free (but some collections have user licenses).

At Vektors, you'll find illustrations from artists around the world and for a wide variety of projects. Illustrations are available in PNG, SVG, Figma, Sketch formats.

At the moment there are 70+ packs.

## Pattern Generator

![](/blog/assets/top-design-resources-jan-2021/img-2.jpg)

[**https://doodad.dev/pattern-generator/#beautiful-dingbats**](https://doodad.dev/pattern-generator/#beautiful-dingbats)

This browser-based pattern generator allows you to create unique royalty-free textures.[‍](https://doodad.dev/pattern-generator/#beautiful-dingbats)

[Pattern Generator](https://doodad.dev/pattern-generator/#beautiful-dingbats) has flexible settings: after choosing a base style, you can then customize your texture using colors, filters and transformations. Each pattern style has its own unique transformations. The Shuffle button acts like a Random button shuffling the styles and outputting randomized items.

You can export the file not only to JPEG or PNG, but also as code for inline SVG and CSS background images.

## Gumaffiliates

![](/blog/assets/top-design-resources-jan-2021/img-3.jpg)

[**https://gumaffiliates.best**](https://gumaffiliates.best/)

‍[Gumaffiliates](https://gumaffiliates.best/) is a new resource that allows you to do affiliate marketing through Gumroad.

Ideal for both creatives without an audience and affiliate marketers with an audience, but without their own products.

By purchasing a product (Gumaffiliates costs $1):

1) Creatives receive a table of people with an audience;

2) Affiliate marketers receive a table with products to promote.

The tables include the following data: product name (for affiliate marketers it's an audience name), niches, website and contact information.

## Parametric Color Mixer

![](/blog/assets/top-design-resources-jan-2021/img-4.jpg)

[**https://colormixer.web.app**](https://colormixer.web.app/02332971ff623007ff9bc9505a440301ffff7c5f55610300/Sunset)

‍[Parametric Color Mixer](https://colormixer.web.app/02332971ff623007ff9bc9505a440301ffff7c5f55610300/Sunset) allows you to generate colors and then customize the palette using interactive equalizers.

Once you have chosen the color, you can adjust it using the following parameters: Position, Impact, Range, Blend.

The number of colors can be either added or reduced.

Exports to SVG and CSS.

## Game UI Database

![](/blog/assets/top-design-resources-jan-2021/img-5.jpg)

[**https://www.gameuidatabase.com**](https://www.gameuidatabase.com/)

‍[This website](https://www.gameuidatabase.com/) has been created specifically for game UI designers.

On this resource, you will find an extensive database of various elements: from pop-up menus to HUD elements. A treasure trove of information for game UI designers.

Nearly 13,000 screenshots from 355 games. More than 80 types of screens.

Game UI Database has a very convenient search system. The sidebar displays filters for screen types: title menus, game states, statistics, information, online features, gameplay and HUD.

You can also filter by genre, theme, camera, game aesthetic, device / controls, etc.

## GIFRun

![](/blog/assets/top-design-resources-jan-2021/img-6.jpg)

[**https://gifrun.com**](https://gifrun.com/)

‍[GIFRun](https://gifrun.com/) allows you to create GIFs without watermarks from different social networks like YouTube, Facebook, TikTok, Vimeo, Twitter and many others that have an embedded video function.

Just insert the source link, define the duration and start time, add text (optional) and your GIF is ready.

## Flow

![](/blog/assets/top-design-resources-jan-2021/img-11.jpg)

[**https://createwithflow.com**](https://createwithflow.com/)

‍[Flow](https://createwithflow.com/) is a new player in the motion design market. The tool's arsenal includes:_‍_

_· Import from Figma, Sketch, XD, Illustrator, Affinity, etc. files (creators promise to add support for Instagram and Dribbble in the future);_

_· A wide range of different functions for cool animation;_

_· Direct export of your Sketch / Figma animations to clean code for Swift, React, SVG, and more._‍

Animations for onboarding screens, buttons, advertising content, and promo videos - Flow will be a great helper in adding some spice to your product.

Plus, the Flow's interface is super intuitive. It incorporates the best practices of the interface from design tools of recent years. Therefore, if you know how to work with Figma or Sketch, then you will quickly figure out how to work with Flow.

## The State of Design Systems 2020

![](/blog/assets/top-design-resources-jan-2021/img-14.jpg)

[**https://material.io/blog/research-state-of-design-systems-2020**](https://material.io/blog/research-state-of-design-systems-2020)

1st place is occupied by the article ["The State of Design Systems: 2020"](https://material.io/blog/research-state-of-design-systems-2020). This is a very informative material where Google has done a research on the use of design systems in 2020. For this study, Google surveyed over 1,000 designers.

Here are the main results.

According to 83% of those surveyed, UI kits are one of the main artifacts of the design system. Which once again confirms the effectiveness of its use.

Figma is a leader in the following spheres:_‍_

_1) Top tools for designing the digital products;_

_2) Top tools for design documentation and hand-off;_

_3) Top tools for managing design systems._‍

Material Design is a leader in the segment of used design systems. Next up are Apple's Human-Interface Guidelines and Bootstrap (unexpected!).

More and more companies create their own design systems. 80% of respondents reported that they have created their own design systems in-house. The main reason is that its own design system better matches the company's brand.
