So I stumbled across this post on Reddit the other day. Some guy spent **18 months** building a design system for free and just open sourced the whole thing. At first I thought, "Alright, another UI kit..." but then I looked closer.

This thing is different.

It's called [**LiftKit**](https://www.figma.com/community/file/1404856652359938563), and it was made by a freelancer who goes by Chainlift. Apparently he got mildly famous in the UI design world last year for a [YouTube video](https://youtu.be/9ElrcTtAxzA?si=_kkrJm9n_uGlILFM) called _The Secret Science of Perfect Spacing_. That video is what kicked off this whole project. Basically, he went down a deep rabbit hole of using the **golden ratio** for literally everything: spacing, font sizes, corner radiuses, all of it.

A design system that just feels right. You know that feeling when a UI just clicks visually? That’s what this is going for.

![UI design component specs balanced with Golden Proportions](/blog/assets/liftkit-open-source-design-system-for-figma-react/img-1.jpg)

Everything is balanced with Golden Proportions

## What is this thing, exactly?

LiftKit is a UI kit that works in **Figma**, and it also comes with **React/Next.js components** that match the designs. So whether you’re designing or coding, you’re covered. But it’s not just about the components. It’s the system behind them that’s cool.

Everything is based on a global scale tied to the golden ratio. No random 12, 14, 20px paddings or whatever. It’s all calculated. You can tell the creator is a bit of a perfectionist in the best way.

And yeah, it’s still early. This is the first official release. But even now, it’s way more polished than some paid kits I’ve used.

## Why you should care (especially if you’re building stuff solo)

If you're like me and constantly juggling between building product, designing the UI, and actually shipping, you don’t want to waste hours fiddling with spacings and layouts. You just want something that looks clean and doesn’t fight you.

Here’s what LiftKit gives you:

-   A ready-to-go **Figma kit** with solid defaults
-   Clean, consistent **React components**
-   Spacing and layout logic that actually makes sense
-   And best of all — it's **free and open source**

## What’s inside?

→ A [Figma file](https://www.figma.com/community/file/1404856652359938563) with all the basics: buttons, forms, nav, modals, etc

→ Components that mirror the Figma stuff in React and Next.js

→ A math-driven spacing system that just works

→ Docs that explain how to use the system and how the design logic works

→ A [GitHub repo](https://github.com/Chainlift/liftkit) if you want to dive into the code or contribute

It’s clearly a passion project. You can feel the care in every layout and pixel.
